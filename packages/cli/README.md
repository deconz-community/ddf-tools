# Command line tool

A command line tool for working with DDFs.

## Installation

```bash
npm install -g @deconz-community/cli
```

## Without install
```bash
npx @deconz-community/cli --help
```

## Usage

### Help

```bash
ddf-tools --help
```

### Validator

```bash
ddf-tools validator --help
ddf-tools validator --directory /path/to/ddf
```
